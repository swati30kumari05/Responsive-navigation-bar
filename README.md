# Responsive Navigation Bar

This project is a simple **responsive** navigation bar created for my Web Technologies lab using HTML, CSS, and JavaScript. It is deployed live using GitHub Pages.

## Features

- Desktop view (width above 800px): menu items appear in a single row, hamburger and close icons are hidden.
- Mobile view (width below 800px): only a hamburger icon is visible at first, the menu slides in from the side when clicked.
- Close icon appears after opening the menu and hides the menu when clicked.
- Layout and colors are inspired by the lab reference image.

## Technologies Used

- HTML for page structure  
- CSS (including media queries) for responsive design and styling  
- JavaScript for opening and closing the mobile menu
